To-Do & Calendar Scheduler
TO-DO LIST CONTROLS: - Check the checkbox to mark a task as done - Tap the X button to remove a task - Red = Unfinished, Green = Finished - Add new tasks at the bottom CALENDAR CONTROLS: - Tap any time slot to pre-fill the event form - Use the arrow buttons to navigate weeks - Use "Jump to date" to go to a specific week - Delete events with the X button TIPS: - Data saves automatically in your browser - Times are in 24-hour format (HH:MM) - Weekly view shows 08:00 to 20:00
